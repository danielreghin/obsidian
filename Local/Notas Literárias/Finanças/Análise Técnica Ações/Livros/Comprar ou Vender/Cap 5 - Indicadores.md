## 5 - Indicadores
São gerados através de gráficos computadorizados que buscam refletir o comportamento médio dos preços. Existem dois tipos:

|   |   |
|---|---|
|**Rastreadores de Tendência**<br><br>Buscam identificar uma tendência|- Médias Móveis<br>- MACD<br>- MACD com sinal<br>- Histograma MACD<br>- Banda de Bollinger|
|**Osciladores**<br><br>Usado quando o mercado está de lado e não há uma tendência clara||

- Rastreadores de Tendência
- Osciladores: Quando mercado de lado  e o preço oscila muito, ajudam a encontrar os níveis de suporte ou resistência.

A lógica é comprar quando os vendedores estão perdendo força e vender quando os compradores estão perdendo força.

### Rastreadores de Tendência

|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Médias Móveis**<br><br>Determina a tendência<br><br>> Faz Alisamento dos ruídos de um gráfico facilitando a determinação de uma tendência. pag, 93<br><br>Por mais que alisa o preço há um atraso. O preço reverte e a média móvel não. Média 10 dias segue o preço mais de perto do que uma média de 20 dias.<br><br>**Média Simples**<br><br>- Peso igual a todos os preços.<br>- Ativos com baixa volatilidade<br><br>**Média ponderada ou exponencial**<br><br>- Peso maior à preços mais recentes<br>- Ativos com alta volatilidade<br>                                                                                                                   |
| **Cruzamento Média X Preço**<br><br>- Tendência de Alta: Preço está acima da média móvel<br>- Tendência de Baixa: Preço está abaixo da média móvel<br>- Comprar: Preço cruzar a média de baixo para cima<br>- Vender: Preço cruzar a média de cima para baixo<br>- Falso cruzamento: Quando a média está na horizontal não há uma tendência definida.<br><br>Quando há uma tendência definida a média e preço se mantém na mesma direção e se distanciam. Mesmo assim ocorrendo penetrações. Se a média não mudar de posição (horizontal x vertical) esses cruzamentos podem ser falsos e aconselhável esperar um pouco.<br>![[Pasted image 20240922211731.png]] |
| **Cruzamento Média Curta X Média Longa**<br><br>Pelo fato da curva de preço ser irregular, pode-se usar outra média para validar os cruzamentos. Evita falso cruzamento. Significa ganho de momentum que é a força necessária para se começar uma tendência.<br><br>- Alta: Média curta está acima da média longa.<br>- Baixa: Média curta está abaixo da média longa<br>- Comprar: Média curta cruzar a média longa de baixo para cima<br>- Vender: Média curta cruzar a média longa de cima para baixo.<br>![[Pasted image 20240922211736.png]]                                                                                                                |
| **MACD**<br><br>Convergência e Divergência de uma média móvel. É a diferença de duas médias móveis. Quando duas médias móveis se aproximam elas tendem a se cruzar a diferença do preço entre elas fica zero. Portanto, usar MACD puro é igual ao cruzamento de duas médias móveis uma de 12 períodos e outra de 26 períodos.<br><br>- Comprar: MACD cruzando negativo para positivo<br>- Vender: MACD cruzando positivo para negativo<br><br>Como o cruzamento pode gerar falsas compras, é sugerido adicionais mais uma média móvel de 9 períodos (Sinal)<br>![[Pasted image 20240922211740.png]]                                                              |
| **MACD com Sinal**<br><br>Como cruzamento pode gerar falsos negativos é sugerido adicionar mais uma média móvel de 9 períodos como sinal.<br><br>- Comprar: MACD cruzar de baixo para cima o sinal<br>- Vender: MACD cruzar de cima para baixo o sinal<br>![[Pasted image 20240922211746.png]]                                                                                                                                                                                                                                                                                                                                                                   |
| **Histograma MACD**<br><br>Análogo a criação do MACD. Resultado da subtração da linha MACD pela linha do sinal.<br><br>- Comprar: Histograma negativo mas ascendente.<br>- Vender: Histograma positivo mas descendente.<br>![[Pasted image 20240922211755.png]]                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Bandas de Bolllinger**<br><br>Linha central é uma média móvel simples. A superior e inferior é duas vezes o desvio padrão. Desvio padrão representa volatilidade. Lógica é que a volatilidade sempre reverte para a média.<br><br>- Comprar: Barra de preço se afasta da banda inferior.<br>- Vender: Barra de preço se afasta da banda superior.<br>- Estreitamento de banda: Rompimento de tendência.<br>![[Pasted image 20240922211759.png]]                                                                                                                                                                                                                |
Como o Histograma MACD contém todas as informações de médias móveis, cruzamentos e MACD é suficiente utilizá-lo. A Bandas de Bollinger é diferente, portanto, pode-se usar em conjunto com o Histograma MACD.
### Osciladores
Sinalizam ponto de reversão de tendência e portanto são utilizados quando o mercado anda de lado. A lógica segue da seguinte forma

Na SobreCompra
- Compradores perdem força
- Oportunidade de vender
- Nunca comprar

Na SobreVenda
- Vendedores perdem força
- Oportunidade para comprar
- Nunca Vender

Para um seguidor de tendência, o oscilador é utilizado de forma secundária e sinaliza o enfraquecimento de uma tendência e o final dela. Mas, a confirmação sempre deve ser feita pelo preço.

| **IFR - Divergência de alta e de baixa**<br><br>Análogo à divergência preço x oscilador conforme mostrado acima<br>- Compra: Preço baixo e oscilador cima<br>- Venda: Preço cima e oscilador baixo<br>![[Pasted image 20240922212056.png]]                                                                                                                                                                                                                                                                                             |
| **IFR - SobreComprado e SobreVendido**<br><br>- Compra: Abaixo da linha inferior e foi para cima<br>- Venda: Acima da linha superior e foi para baixo<br>![[Pasted image 20240922212102.png]]                                                                                                                                                                                                                                                                                                                                          |
| **Momento**<br><br>Mede se uma tendência está acelerando ou se os preços movem rápido ou devagar.<br>É uma linha flutuante que oscila, por isso, é um oscilador.<br>![[Pasted image 20240922212108.png]]                                                                                                                                                                                                                                                                                                                               |
| **Estocástico**<br><br>Relaciona o preço de fechamento com os máximos e mínimos recentes. Quando os preços sobem o fechamento tendem a ser aproximar da máxima do dia e vice versa.<br><br>Tem duas linhas %K e %D (média móvel de %K)<br><br>Quando o fechamento se aproxima do maior máximo, %K se aproxima de 100. Quando o fechamento se aproxima do maior mínimo, o %L se aproxima de 0.<br><br>- Compra: %K cruzar o %D de baixo para cima<br>- Venda: %K cruzar o %D de cima para baixo<br>![[Pasted image 20240922212116.png]] |
| **Estocástico - Divergência de alta baixa**<br><br>Segundo o George Lane, a sinalização mais importante é a divergência do %D em relação ao gráfico de preço.<br><br>Vide divergência preço x oscilador para detalhes<br>![[Pasted image 20240922212123.png]]                                                                                                                                                                                                                                                                          |
| **Estocástico - SobreCompra e SobreVenda**<br><br>Quando %K se aproxima dos extremos há o sinal de SobreCompra e SobreVenda. Esse sinal é prioritário em relação à regra de cruzamento.<br>![[Pasted image 20240922212129.png]]                                                                                                                                                                                                                                                                                                        |
| **OBV - On Balance Volume**<br><br>Associa a evolução dos preços com os volumes negociados a cada dia. Tenta medir os níveis de acumulação ou de distribuição.<br><br>O volume é adicionado se o fechamento for de alta e subtraído se for de baixa. Se não variar, não há ajuste<br><br>Se o mercado está andando de lado, observe o OBV<br><br>- Se subindo: sinal de rompimento para alta<br>- Se caindo: sinal de rompimento para baixa<br>![[Pasted image 20240922212135.png]]<br><br>![[Pasted image 20240922212142.png]]        |
