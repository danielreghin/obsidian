Existem três escalas menores são elas:
- [[Escala menor natural]]
- [[Escala menor harmônica]]
- [[Escala menor clássica]]

Em resumo os intervalos das escalas são como abaixo:


| Escalas Menors    |  I  | II  |   III   | IV  |  V  |   VI    |    VII     |    I    |
| ----------------- | :-: | :-: | :-----: | :-: | :-: | :-----: | :--------: | :-----: |
| Natural           |  -  | tom | semitom | tom | tom | semitom |    tom     |   tom   |
| Harmônica         |  -  | tom | semitom | tom | tom | semitom | tom e meio | semitom |
| Melódica Real     |  -  | tom | semitom | tom | tom |   tom   |    tom     | semitom |
| Melódica Clássica |     |     |         |     |     |         |            |         |
A melódica clássica, desce como real e sobe como natural.
## Acordes mais usados
Na [[Harmonia Funcional]], existem alguns acordes mais usados nessa escala.

As escalas podem ser usadas como empréstimo modal da escala maior e também entre escalas menores, com isso, existem alguns acordes mais usados, são eles:
- N = Escala Menor Natural
- H = Escala Menor Harmônica
- M = Escala menor Melódica
